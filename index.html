rfcshow by brickviking

Fetches, stores, searches and displays RFC documents

This project does like the description says, and like the README says. Aim is:

  • fetch documents from remote sites
  • store them locally (preferably compressed)
  • provide search functions
  • display required documents
  • update documents as new ones are found

Dependencies

Perl, Curses, Curses::UI, Compress::Bzip2 (optional), File::Slurp, Getopt::Long

Install

chmod +x rfcshow ./rfcshow
